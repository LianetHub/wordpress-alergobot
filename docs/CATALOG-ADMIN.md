# Каталог оборудования — инструкция по настройке в админке

## 1. Плагины и базовая настройка

1. Установите и активируйте **Advanced Custom Fields Pro** и **Contact Form 7**.
2. Активируйте тему **alergobot** (или переключите повторно) — создадутся категории каталога.
3. **ACF → Tools → Import** — импортируйте три файла из папки темы `acf-json/`:
   - `group_catalog_page.json` — поля страницы каталога
   - `group_product_category.json` — поля категорий
   - `group_product_single.json` — поля товаров
4. **Настройки → Постоянные ссылки → Сохранить** (обязательно после деплоя).
5. В **Настройки сайта** (ACF Options) заполните шорткоды CF7:
   - `cf7_zakaz` — форма заказа
   - `cf7_konsultaciya` — форма консультации
   - `cf7_prezentaciya` — форма заявки на презентацию (каталог)

## 2. Страница каталога

| Параметр | Значение |
|----------|----------|
| Заголовок | Каталог оборудования и реагентов |
| Slug | `oborudovanie` |
| Шаблон | **Каталог** |

Заполните ACF-поля группы «Каталог — страница»:

- шапка (заголовок, текст);
- теги навигации выводятся автоматически из категорий каталога (`product_category`);
- 2 секции каталога (устройства / панели и реагенты);
- блок заявки (заголовок, текст; форма подтягивается из CF7);
- FAQ.

Если поля пустые — соответствующий блок на странице не выводится.

Старый slug `/katalog/` при необходимости настройте редирект на `/oborudovanie/` отдельно (Rank Math / Redirection).

## 3. Категории каталога

**Продукция → Категории каталога**

| Название | Slug | URL |
|----------|------|-----|
| Оборудование | `ustroystva` | `/oborudovanie/ustroystva/` |
| Анализаторы | `analizatory` | `/oborudovanie/analizatory/` |
| Реагенты | `reagenty` | `/oborudovanie/reagenty/` |
| Панели | `paneli` | `/oborudovanie/paneli/` |

Для каждой категории заполните ACF-группу «Каталог — категория»:

- шапка (заголовок, текст, логотип);
- блок «Как работает» (заголовок, тег, текст, галерея);
- слайдер преимуществ;
- CTA-блок.

Список товаров выводится автоматически из записей CPT **Продукция**, привязанных к категории.

URL товара: `/oborudovanie/{slug}/`

### Для каждого товара заполните

1. **Заголовок** и **slug** (см. таблицу).
2. **Цитата (Excerpt)** — краткое описание для карточки в категории.
3. **Миниатюра** — изображение для карточки и hero (если не задано отдельное ACF-поле).
4. ACF-группа «Каталог — товар»:
   - шапка (текст, наличие, рейтинг, изображение);
   - вкладка «Описание» (лид, блоки списков, таблица аллергенов для панелей);
   - вкладка «РУ / КТРУ» (характеристики, файл РУ);
   - вкладка «Видео» (постер, URL);
   - вкладка «Документы»;
   - блок «Преимущества PROTIA»;
   - «Смотрят также» (relationship).

Эталон контента — статическая вёрстка Q-SMART в `src/product.html`.

## 5. Меню и SEO

- Пункт «Каталог» в меню (ACF Options → `glavnoe_menyu`) → `/oborudovanie/`
- Удалите или не публикуйте отдельную страницу `/analizatory/` — категория доступна по `/oborudovanie/analizatory/`
- Проверьте sitemap и canonical после flush permalinks

## 6. Проверка после наполнения

- [ ] `/oborudovanie/` — страница каталога
- [ ] `/oborudovanie/analizatory/` — список анализаторов
- [ ] `/oborudovanie/analizator-immunologicheskiy-q-smart/` — карточка товара
- [ ] Форма заявки на каталоге отправляется через CF7
- [ ] Попапы «Заказать» / «Задать вопрос» на категории и товаре работают
- [ ] Блок «Смотрят также» на товаре
